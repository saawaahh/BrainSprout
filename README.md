# BrainSprout


## A journal app designed with recovery in mind

Our team implemeted a journal app designed to help stroke victims throughout their recovery process. This was created for our CS4326 semester project. This course was conducted by  Dr. Samantha Aziz

If you are not familiar with the different technologies used in this project, please refer to the respective docs.

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
